# Sentiment Analysis (Simple RNN)

This folder contains notebooks and scripts for a simple sentiment analysis example using a basic RNN in Keras/TensorFlow.

Contents

- `sentiment_analysis_simplernn.ipynb` — main notebook demonstrating integer encoding, RNN model building, training, and evaluation.
- `integer_encoding_simplernn.ipynb` — helper notebook showing integer encoding and preprocessing steps.

How to run

1. Open `sentiment_analysis_simplernn.ipynb` in Jupyter or VS Code.
2. Ensure you have TensorFlow installed (e.g., `pip install tensorflow`).
3. Run cells sequentially to preprocess data, build the model, train, and evaluate.

Notes

- The notebook is self-contained and documents assumptions about dataset format and dependencies.
- If you want this folder pushed to a remote GitHub repo, ensure your local Git has authentication configured (SSH key or PAT).
